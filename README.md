# ioIndustry

A landing page do ioIndustry foi desenvolvida com Vue 3 e Tailwind CSS, apresentando um design moderno, responsivo e focado em performance.

O site destaca as principais funcionalidades da **plataforma ioIndustry** — uma solução para monitorização, gestão e otimização industrial — através de seções bem estruturadas que incluem:

- Apresentação do produto e suas principais vantagens;

- Módulos e diferenciais com cards interativos;

- FAQ dinâmico com sistema de acordeão;

- Carrossel de imagens demonstrando a aplicação;

- Seção de clientes e contactos;

- Questionário interativo, que avalia o nível de digitalização da empresa e gera resultados personalizados.

O layout foi desenvolvido com foco em usabilidade, acessibilidade e adaptação para dispositivos móveis, oferecendo uma navegação fluida e uma experiência visual consistente.
